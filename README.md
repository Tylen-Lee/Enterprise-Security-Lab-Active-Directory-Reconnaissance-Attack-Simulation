# Enterprise-Security-Lab-Active-Directory-Reconnaissance-Attack-Simulation

## Objective

The objective of this project is to design and configure a secure enterprise-like environment by implementing Active Directory services, DNS configurations, and mail services within a virtual network. The lab further aims to simulate real-world cybersecurity scenarios by performing network reconnaissance, vulnerability assessment, and controlled attack techniques.

Using tools such as Nmap, SET Toolkit, and Burp Suite, the project explores key phases of the attack lifecycle including information gathering, initial access, and credential exploitation. The goal is to develop a practical understanding of how systems are configured, how vulnerabilities can be identified, and how attacks can be executed and analysed in a controlled and ethical environment.

Additionally, this lab reinforces the importance of security best practices, user awareness, and defensive strategies in protecting enterprise systems.

I further mapped these activities to the MITRE ATT&CK framework, linking reconnaissance (Nmap scanning) and social engineering techniques to known adversary tactics and techniques.

## Key Take-aways

- Understanding of Active Directory and domain environments  
- DNS configuration including MX records and reverse lookup zones  
- Mail server setup and client configuration  
- Network reconnaissance using Nmap (ping sweep, full scan)  
- Social engineering concepts through controlled phishing simulation  
- Credential harvesting techniques and associated risks  
- Web application testing and brute force attack concepts using Burp Suite  
- Importance of firewall rules and port management  
- Real-world attack flow from reconnaissance to exploitation

 ## Skills Learned

- Active Directory setup and user management  
- DNS configuration and troubleshooting  
- Network scanning and enumeration  
- Social engineering awareness and simulation  
- Web application security testing  
- Firewall configuration and port management  
- Analytical thinking and attack flow understanding  
- Security testing in a controlled lab environment  

## Tools & Technologies

- Kali Linux  
- Windows Server / Windows 10
